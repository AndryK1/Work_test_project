# Тестовый проект

Приветствую вас в этом тестовом проекте!

## О проекте

Всё реализовано согласно полученному ТЗ.

## Особенности реализации

### Изображения
Из-за отсутствия в моковых данных ссылок на изображение и соответвующих полей в entity:
- Все изображения карточек 
- Логотипы компаний 
в настоящее время заменены на плейсхолдеры.

### Хранение данных
- Реализована запись в локальную БД с помощью **Room**

### Экран профиля
- Имеется рабочий **RecyclerView**
- Поскольку по ТЗ это экран-заглушка, туда передаются **mock данные**
- У экрана отсутствует ViewModel (согласно требованиям)

### Навигация
Навигация реализована через **Koin**.

### Экран регистрации
Создан экран регистрации, но в данный момент он является **заглушкой**:
- Логики на нём нет
- По ТЗ его реализация не требовалась
